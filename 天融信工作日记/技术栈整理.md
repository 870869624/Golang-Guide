- 路由组分离公开路由和认证路由（`/auth/*`）

| structpb | - | 动态结构体构建，用于通用任务下发参数 |

### gRPC 使用模式

**客户端流式（Client Streaming）** — NGDLP 敏感文件上传：
```protobuf
rpc UploadFileSensitive(stream UploadSensitiveFileReq) returns (Response)
```
Agent分片发送文件，服务端循环 `stream.Recv()` 直到 EOF。

**服务端流式（Server Streaming）** — NGDLP 文件下载：
```protobuf
rpc DownloadSensitiveFile(stream DownloadSensitiveFileReq) returns (stream DownloadSensitiveFileResponse)
```

**一元RPC（Unary）** — NGEDR 任务下发、资产查询等常规调用。

### gRPC 客户端管理

NGEDR 项目注入了 **16个gRPC客户端**：
| 客户端 | 目标服务 | 用途 |
|--------|---------|------|
| Agent | 终端Agent服务 | 任务下发、策略同步 |
| Asset | 资产服务 | 终端元信息查询（在线状态、OS、负责人） |
| RolesvcRPC | 角色权限服务 | RBAC权限查询（部门树递归） |
| UsersvcRPC | 用户服务 | 用户信息查询 |
| SystemsvRPC | 系统配置服务 | 系统配置管理 |
| SyslogsvcRPC | 系统日志服务 | 操作日志记录 |
| BFF | BFF网关 | 认证鉴权 |
| LogminerRPC | 日志挖掘服务 | ClickHouse查询代理 |
| ErsvcRPC | 事件路由服务 | 日志平台对接 |
| LicenseRPC | 许可证服务 | License管理 |
| ManagerRPC | 管理服务 | DIP管理 |
| ThreatRPC | 威胁文件服务(自服务) | 威胁文件管理 |
| SoftwaremngRPC | 软件管理(自服务) | 终端软件管理 |
| PatchmngRPC | 补丁管理(自服务) | 补丁管理 |
| DynamicauthRPC | 动态认证(自服务) | 动态认证 |
| MsdmngRPC | 移动存储管理(自服务) | U盘/移动设备管理 |

### gRPC 配置
- mTLS 双向证书认证（CA证书 + 服务证书 + 私钥）
- `grpc.MaxRecvMsgSize` 限制最大消息（默认10MB）
- 客户端配置：Trace=true, Duration=true, Breaker=true, Timeout=true
- 非阻塞模式：`nonblock: true`

## 四、数据存储

### 4.1 MongoDB

| 技术 | 版本 | 使用项目 |
|------|------|---------|
| go.mongodb.org/mongo-driver | v1.17.x | 天融信 + 城汇玩(间接) |

**NGDLP 使用方式**：
- 主数据库，存储敏感文件元数据、审批记录、采集配置、列配置
- go-zero `mon.MongoModel` 自动生成 CRUD
- 自定义批量写入器（`FileBatchWriter`）：
  - 缓冲通道（buffer=500），批量大小50条，10秒强制flush
  - 策略模式：`DefaultMergeStrategy`（按MD5合并） / `AutoUploadMergeStrategy`（按FileID覆盖）
- `SensitiveFileUpdateQueue`：全局单例，缓冲2000任务，单worker串行消费

**NGEDR 使用方式**：
- 主数据库，21个集合
- 索引策略：按 `tenant_id + 时间/分组/病毒名/IP/MD5` 建复合索引
- 聚合管道：`$match` → `$group` → `$sort`
- `BulkWrite` 无序批量更新（`SetOrdered(false)`）
- 动态间隔聚合：根据上次耗时自动调整（1分钟/5分钟/15分钟/30分钟）

**城汇玩使用方式**：
- GORM AutoMigrate 映射 100+ 个数据模型
- golang-migrate 管理 65 个迁移版本（130个SQL文件）

## 八、加密与安全

| 技术 | 说明 | 使用项目 |
|------|------|---------|
| AES-256-GCM | 文件加密存储（分块加密，每块用块索引作AAD） | 天融信 NGDLP |
| crypto/rand | GCM Nonce随机生成（12字节） | 天融信 NGDLP |
| mTLS | gRPC双向证书认证 | 天融信 |
| AK/SK签名 | 级联平台间通信认证 | 天融信 |
| JWT (golang-jwt/jwt/v4) | 用户认证Token | 城汇玩 |
| 国密SM2/SM3/SM4 | 敏感数据加密 | 天融信/城汇玩 |

### AES-GCM 加密文件格式
```
[Nonce 12字节] [分块数 4字节 BigEndian] [密文长度1 4字节] [密文1] [密文长度2 4字节] [密文2] ...
```
- 密钥：文件MD5值（主）或FileID（兼容旧版）
- Nonce写入文件头部（明文）
- 每分块用块索引作AAD，确保加密上下文唯一
