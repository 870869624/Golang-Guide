## 二、Kubernetes 相关命令

### 2.1 服务管理

#### 查看所有服务
```bash
kubectl get svc -A
```

#### 查看指定命名空间的服务
```bash
kubectl get svc -n sdsp
```

#### 查看指定命名空间的服务（详细信息）
```bash
kubectl get services --all-namespaces
```

### 2.2 Pod 管理

#### 查看指定命名空间的 Pod
```bash
kubectl get pods -n sdsp
```

#### 进入 Pod 内部
```bash
kubectl exec -it virt-launcher-clean-node01-hv4sn -- /bin/bash
kubectl exec -it cleansvc-7dc5b6d855-rrfzc -n sdsp -c cleansvc -- sh
kubectl exec -it -n sdsp  cleansvc-5858f89d9d-8txkl   -- /bin/bash
```

#### 查看 Pod 日志
```bash
kubectl logs -f agcbb-7787c8dcb7-24r82 -n sdsp
```

#### 查看 Pod 监听的端口
```bash
kubectl exec -it node-7b9f8bcd9-4xk6m -n sdsp -- /bin/sh -c "ss -tunlp"
```

#### 查看 Pod 镜像拉取策略
```bash
kubectl get pod cleansvc-7d8cf5c4f4-xq6xl -n sdsp -o yaml | grep -A 5 "imagePullPolicy"
```

#### 查看 Pod 详细信息
```bash
kubectl describe pod cleansvc-7d8cf5c4f4-xq6xl -n sdsp
```

### 2.3 部署管理

#### 查看 Deployment/StatefulSet/DaemonSet 列表
```bash
kubectl get deployments,statefulsets,daemonsets -n topsec-ngdlp
```

#### 查看 Deployment 详细信息

```bash

```

#### 端口转发
```bash
kubectl port-forward svc/agcbb 15555:15555 -n sdsp
```

#### 新增 Pod 的 NodePort
```bash

  --type=NodePort --port=27017 --target-port=27017
```

## 三、MongoDB 相关命令

### 3.1 连接与操作

#### 进入 MongoDB
```bash
mongo -u admin -p 'Cloud@talent123!@#'
```

#### 切换数据库
```bash
use 数据库名
```

#### 查看集合数据
```bash
db.集合名称.find({"_id":ObjectId("68dba608b9215cd46fb12bf4")}).limit(10)
```

#### 统计集合数据
```bash
db.file_record.countDocuments({})
```

#### 根据字段统计集合数据
```bash
db.file_record.aggregate([
  {
    $group: {
      _id: {
        clean_task_id: "$clean_task_id",
        channel_id: "$channel_id"
      },
      count: { $sum: 1 }
    }
  },
  { $sort: { count: -1 } }
])
```

### 3.2 索引管理

#### 查看索引
```bash
kubectl exec -it -n topsec-topngdlp deployments/mongodb  -- sh  -c 'mongo sensitive -uroot -pTopsecCdu_1130 --authenticationDatabase admin --eval  "db.file.getIndexes()" '
```

#### 创建索引
```bash
kubectl exec -it -n topsec-topngdlp deploy/mongodb  -- mongo sensitive -uroot -pTopsecCdu_1130 --authenticationDatabase admin --eval 'db.file.createIndex({tenant_id:1,file_id:1,platform:1})'
```

### 3.3 数据管理

#### 删除指定时间段的数据
```bash
kubectl exec -it -n topsec-topngdlp deploy/mongodb  -- mongo sensitive -uroot -pTopsecCdu_1130 --authenticationDatabase admin --eval 'db.file.deleteMany({created_at:{$lt:1768733424}})'
```

#### 统计指定时间段的数据
```bash
kubectl exec -it -n topsec-topngdlp deploy/mongodb  -- mongo sensitive -uroot -pTopsecCdu_1130 --authenticationDatabase admin --eval 'db.file.countDocuments({created_at:{$lt:1768733424}})'
```

#### 获取索引大小
```bash
kubectl exec -it -n topsec-topngdlp deployments/mongodb -- sh -c 'mongo -uroot -pTopsecCdu_1130 --authenticationDatabase admin --eval "
db.adminCommand({listDatabases:1}).databases.forEach(function(database) {
    db = db.getSiblingDB(database.name);
    if(database.name != \"asset\") {
     return;
    }
    db.getCollectionNames().forEach(function(collection) {
        var stats = db[collection].stats();
        print(database.name + \".\" + collection +
              \" - 数据大小: \" + Math.round(stats.storageSize / 1024 / 1024) + \"MB, \" +
              \"索引大小: \" + Math.round(stats.totalIndexSize / 1024 / 1024) + \"MB\");
    });
});
"'
```
