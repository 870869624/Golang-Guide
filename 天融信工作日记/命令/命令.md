## 二、Kubernetes 相关命令

### 2.1 服务管理

#### 查看所有服务
```bash
kubectl get svc -A
```

#### 查看指定命名空间的服务
```bash
kubectl get svc -n sdsp
```

#### 查看指定命名空间的服务（详细信息）
```bash
kubectl get services --all-namespaces
```

#### 关闭 ngdlp（彻底停止）
```bash
kubectl scale deployment ngdlp --replicas=0 -n topsec-topngdlp
```

#### 删除 Pod
```bash
kubectl delete pod -n sdsp ngdlp-7d8cf5c4f4-xq6xl
```

#### ✅ 验证关闭状态
```bash
kubectl get pod -n topsec-topngdlp | grep ngdlp
# 应该看不到 ngdlp 的 Pod（或者显示 Terminating 后消失）
```

#### 🔁 重新启动（恢复）
```bash
kubectl scale deployment ngdlp --replicas=1 -n topsec-topngdlp
# ✅ 或者滚动重启
kubectl rollout restart deployment ngedr -n sdsp
# ✅ 检查状态
kubectl get pod -n topsec-topngdlp | grep ngdlp
```

#### ✅ 停止命名空间下所有 Deployment（最常用）
```bash
kubectl scale deployment --all --replicas=0 -n topsec-topngdlp
```

#### 🔄 停止其他类型的工作负载 and 重启
```bash
# 如果环境中还有 StatefulSet（如 mongodb-0、redis-0、clickhouse-0 等）、DaemonSet（如 metrics-node-lz2bx）等资源，需要分别处理：
# 停止所有 StatefulSet
kubectl scale statefulset --all --replicas=0 -n topsec-topngdlp

# 停止所有 DaemonSet（注意：DaemonSet 通常不能缩为 0，但可以设置 nodeSelector 或直接删除，更简单的是 kubectl delete daemonset --all -n ...）
kubectl delete daemonset --all -n topsec-topngdlp   # 彻底删除，恢复需重新创建

# ⚠️ 重要提醒
# 不要用 kubectl delete pod --all：删除所有 Pod 会被 Deployment/StatefulSet 立即重建，达不到“彻底关闭”的效果。

# --all 的作用范围：只作用于当前命令指定的资源类型。没有 --all 就能跨类型停止所有资源的参数。

# 恢复方法：重新设置副本数为原来的值（例如 1）即可：
kubectl scale deployment --all --replicas=1 -n topsec-topngdlp
```

#### 获取索引大小
```bash
kubectl exec -it -n topsec-topngdlp deployments/mongodb -- sh -c 'mongo -uroot -pTopsecCdu_1130 --authenticationDatabase admin --eval "
db.adminCommand({listDatabases:1}).databases.forEach(function(database) {
    db = db.getSiblingDB(database.name);
    if(database.name != \"asset\") {
     return;
    }
    db.getCollectionNames().forEach(function(collection) {
        var stats = db[collection].stats();
        print(database.name + \".\" + collection +
              \" - 数据大小: \" + Math.round(stats.storageSize / 1024 / 1024) + \"MB, \" +
              \"索引大小: \" + Math.round(stats.totalIndexSize / 1024 / 1024) + \"MB\");
    });
});
"'
```
