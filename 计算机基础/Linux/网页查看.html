核心命令速查表

文件操作命令

命令 描述 示例 操作
ls 列出目录内容 ls -la
cd 切换目录 cd /home/user
mkdir 创建目录 mkdir -p /path/to/dir
cp 复制文件/目录 cp -r source dest
rm 删除文件/目录 rm -rf directory
mv 移动/重命名 mv old.txt new.txt

系统管理命令

服务管理

systemctl start nginx
systemctl stop nginx
systemctl status nginx
systemctl enable nginx

用户管理

useradd username
passwd username
id username
userdel -r username

系统配置指南

网络配置

修改IP地址

vim /etc/sysconfig/network-scripts/ifcfg-ens33 # 修改后重启网络 service network restart

配置主机名

# 查看主机名 hostname # 修改主机名 vim /etc/hostname # 配置hosts映射 vim /etc/hosts

防火墙配置

基本操作

# 查看状态 systemctl status firewalld # 关闭防火墙 systemctl stop firewalld # 开放端口 firewall-cmd --add-port=80/tcp --permanent # 重载配置 firewall-cmd --reload

实践练习

基础练习

练习1：创建目录结构
mkdir -p ~/projects/web/{html,css,js} cd ~/projects/web ls -la
练习2：文件操作
touch index.html style.css script.js cp index.html backup.html mv script.js js/app.js

系统管理练习

练习3：用户管理
sudo useradd devuser sudo passwd devuser id devuser su - devuser
练习4：权限设置
chmod 755 ~/projects chown devuser:devuser ~/projects ls -la ~/projects

快捷参考卡片

文件权限

7 = rwx
6 = rw-
5 = r-x
4 = r--

快捷键

Ctrl+C: 中断
Ctrl+D: 退出
Ctrl+Z: 暂停
Tab: 自动补全

常用路径

/etc: 配置
/var/log: 日志
/home: 用户
/usr/bin: 命令
